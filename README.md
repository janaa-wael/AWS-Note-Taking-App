# 📝 Complete Note Taking System

A full-stack note-taking application with user authentication, categories, archiving, and dark mode.

## 🚀 Features

- **User Authentication** - Register/Login with JWT
- **Create, Read, Update, Delete Notes**
- **Categories** - Organize notes by custom categories
- **Archive** - Move notes to archive instead of deleting
- **Dark Mode** - Toggle between light and dark themes
- **Responsive Design** - Works on desktop and mobile
- **RESTful API** - Well-structured backend API

## 🛠️ Tech Stack

### Backend
- FastAPI (Python)
- PostgreSQL
- SQLAlchemy ORM
- JWT Authentication
- Bcrypt Password Hashing

### Frontend
- React.js
- Tailwind CSS
- Axios for API calls
